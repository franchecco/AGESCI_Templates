# AGESCI Templates ⚜️
Questa è una pagina Github con vari template che possono aiutare i gruppi d'Italia a creare le loro grafiche, con uno stile accattivante.

***Il tutto OPEN-SOURCE***

Ogni collaborazione e aiuto è apprezzato

## Template per Sito Web

Il sito è strutturato in modo tale da permettere ad ogni gruppo di creare, o tramite pagine preimpostate o con blocchi, il proprio sito web.

Le pagine preimpostate saranno modificabili semplicemente modificando un file JSON, che inserirà i testi e le immagini nelle relative posizioni, permettendo di creare quante pagine si vogliono
